# منصة حركة - نظام إدارة التربية البدنية
# Haraka Platform - Physical Education Management System

## 🎯 نظرة عامة

منصة حركة هي نظام شامل لإدارة التربية البدنية في المدارس السعودية، مع نظام أمان متقدم يضمن حماية البيانات وفقاً لأعلى المعايير العالمية.

## 🔒 نظام الأمان

### Row Level Security (RLS)
- **25+ سياسة أمان** للواجهات التسع
- **عزل كامل للبيانات** - كل مستخدم يرى بياناته فقط
- **Views مجهولة** للوزارة والمديريات بدون بيانات شخصية

### تسجيل العمليات (Audit Logging)
- **تسجيل تلقائي** لجميع العمليات
- **كشف الأنشطة المشبوهة** والتنبيهات الفورية
- **تتبع محاولات الاختراق** مع التسجيل الشامل

## 🚀 التثبيت السريع

### المتطلبات
- Docker & Docker Compose
- 4GB RAM على الأقل
- 10GB مساحة تخزين

### الإعداد
```bash
# 1. استنساخ المشروع
git clone <repository-url>
cd haraka-platform

# 2. تشغيل سكريبت الإعداد
chmod +x scripts/setup.sh
./scripts/setup.sh

# 3. تشغيل الاختبارات
chmod +x scripts/test.sh
./scripts/test.sh
```

## 🔐 الواجهات والصلاحيات

### 1. الطالب/الشاب
- ✅ يرى بياناته الشخصية فقط
- ✅ يرى جلسات التمارين الخاصة به
- ❌ لا يمكنه الوصول لبيانات الآخرين

### 2. ولي الأمر
- ✅ يرى بيانات أطفاله المسجلين فقط
- ✅ يرى تقارير أطفاله
- ❌ لا يمكنه الوصول لأطفال آخرين

### 3. المعلم/المدرب
- ✅ يرى بيانات طلاب صفه فقط
- ✅ يمكنه إنشاء جلسات وتقارير
- ❌ لا يمكنه الوصول لصفوف أخرى

### 4. مديرية التربية
- ✅ ترى إحصائيات منطقتها مجمعة ومجهولة
- ✅ تقارير أداء المدارس (بدون أسماء)
- ❌ لا ترى بيانات شخصية للطلاب

### 5. الوزارة
- ✅ ترى إحصائيات وطنية مجمعة ومجهولة فقط
- ✅ اتجاهات الأداء العامة
- ❌ لا ترى أي بيانات شخصية (PII)

### 6. الأدمن
- ✅ وصول كامل مع تسجيل شامل
- ✅ إدارة النظام والمستخدمين
- 📝 جميع العمليات مُسجلة تلقائياً

## 🔧 الإعداد المتقدم


## 📊 المراقبة والتنبيهات

### لوحات المراقبة
- **Grafana**: http://localhost:3002
- **pgAdmin**: http://localhost:8080

### التنبيهات الأمنية
- محاولات وصول مرفوضة متكررة (5+ في الساعة)
- استخدام عناوين IP متعددة (3+ لنفس المستخدم)
- أنشطة غير طبيعية

### تقارير الأمان
```bash
# عرض الأنشطة المشبوهة
SELECT * FROM suspicious_activities;

# إحصائيات المراجعة
SELECT * FROM admin_audit_statistics WHERE audit_date >= CURRENT_DATE - INTERVAL '7 days';

# محاولات الوصول المرفوضة
SELECT * FROM security_access_denied_log WHERE attempts_from_same_ip_per_hour >= 3;
```

## 🛠️ الصيانة

### النسخ الاحتياطية
```bash
# نسخ احتياطي لقاعدة البيانات
docker-compose exec postgres pg_dump -U postgres haraka_db > backup_$(date +%Y%m%d).sql

# استعادة النسخة الاحتياطية
docker-compose exec -T postgres psql -U postgres haraka_db < backup_file.sql
```

### تنظيف السجلات القديمة

```bash
# حذف سجلات المراجعة الأقدم من سنة
```

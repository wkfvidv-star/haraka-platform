# قاموس المصطلحات العربية الجزائرية (Arabic Glossary)

## نظرة عامة
هذا القاموس يحتوي على جميع المصطلحات المستخدمة في منصة تتبع اللياقة البدنية، مع التركيز على التوافق الثقافي واللغوي مع الجمهور الجزائري.

## 7. إرشادات الاستخدام (Usage Guidelines)

### قواعد الكتابة (Writing Rules)
1. **استخدام الأرقام العربية**: ١٢٣٤٥٦٧٨٩٠
2. **التاريخ**: يوم/شهر/سنة (٠٢/١٠/٢٠٢٤)
3. **الوقت**: نظام 24 ساعة (١٤:٣٠)
4. **العملة**: دينار جزائري (د.ج)
5. **الوحدات**: النظام المتري (متر، كيلوغرام، لتر)

### التنسيق النصي (Text Formatting)
- **العناوين**: خط عريض وحجم أكبر
- **النصوص المهمة**: لون مميز أو تسطير
- **الروابط**: لون أزرق مع تسطير
- **رسائل الخطأ**: لون أحمر مع رمز تحذير
- **رسائل النجاح**: لون أخضر مع رمز صح

### الاختصارات المقبولة (Acceptable Abbreviations)
| الاختصار | المعنى الكامل | الاستخدام |
|----------|---------------|-----------|
| د.ج | دينار جزائري | العملة |
| كغ | كيلوغرام | الوزن |
| سم | سنتيمتر | الطول |
| ن/د | نبضة في الدقيقة | معدل القلب |
| س.ح | سعرة حرارية | الطاقة |

## 8. قائمة التحقق النهائية (Final Checklist)

### ✅ التحقق من التسميات
- [ ] جميع أسماء الأدوار متسقة عبر النظام
- [ ] المصطلحات التقنية مشروحة عند الحاجة
- [ ] أسماء الولايات صحيحة ومكتملة (58 ولاية)
- [ ] العبارات مهذبة ومناسبة ثقافياً
- [ ] الأرقام والتواريخ بالتنسيق العربي

### ✅ التحقق من الاتساق
- [ ] نفس المصطلح يُستخدم لنفس المفهوم
- [ ] لا تضارب في المعاني
- [ ] التدرج الهرمي واضح (تلميذ ← ولي ← أستاذ ← مدير)
- [ ] المستويات الإدارية محددة بوضوح

### ✅ التحقق الثقافي
- [ ] احترام التقاليد الجزائرية
- [ ] مناسب لجميع الأعمار
- [ ] يراعي الفروق بين الجنسين
- [ ] متوافق مع النظام التعليمي الجزائري
- [ ] يحترم الهوية الوطنية

### ✅ التحقق التقني
- [ ] دعم RTL كامل
- [ ] الخطوط العربية واضحة
- [ ] التنسيق صحيح على جميع الأجهزة
- [ ] الترجمة دقيقة ومفهومة
- [ ] لا أخطاء إملائية أو نحوية

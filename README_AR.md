# تعليمات تشغيل منصة حركة (Haraka Platform)

لحل مشكلة الاتصال بالخادم (Backend) وضمان عمل جميع الخصائص (مثل إنشاء الحساب وتسجيل الدخول)، يرجى اتباع الخطوات التالية:

## التشغيل الكامل (الخادم + الواجهة)

لقد قمت بإضافة أمر جديد يقوم بتشغيل السرفر والواجهة معاً في وقت واحد. استخدم هذا الأمر دائماً عند التطوير:

```bash
npm run dev:all
```

أو إذا كنت تستخدم `pnpm`:

```bash
pnpm run dev:all
```

## ملاحظات هامة

1. **المنفذ 3001**: السرفر الخلفي يعمل على المنفذ 3001. تأكد من عدم وجود برامج أخرى تستخدم هذا المنفذ.
2. **قاعدة البيانات**: السرفر متصل حالياً بقاعدة بيانات Supabase الموضحة في ملف `.env`.
3. **التسجيل**: بعد تشغيل الأمر أعلاه، يمكنك الذهاب إلى صفحة إنشاء الحساب وستعمل عملية الربط بنجاح.
